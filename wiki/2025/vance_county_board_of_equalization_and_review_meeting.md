# Vance County Board of Equalization and Review Meeting

**Date:** June 3, 2025  
**Year:** 2025  
**Type:** Board of Commissioners Meeting Minutes

## Summary

The Vance County Board of Equalization and Review held a special session to address taxpayer appeals regarding property valuations. Several property values were reduced based on appraisals and comparable property values provided by the appellants. Some appeals were continued to allow further review by the tax office, and a few were upheld due to lack of evidence or absence of the appellant.

## Key Decisions

- Reduced property value for Shirley Julian to $151,840.
- Reduced property value for Michael Epps to $141,887.
- Reduced property value for Arnold Briggs to $502,000.
- Reduced property value for Thomas Franklin to $273,231.
- Reduced property value for Thomas & Jamie Doran to $340,000.
- Reduced property value for Crystal Cash to $103,074.
- Reduced property value for Patsy Briggs to $253,778.
- Reduced property value for Russell Newman to $365,000.
